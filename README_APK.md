# APK Oluşturma Talimatları

## Android Customer App
1. C:\FunBreakVale_Android_Customer klasörüne gidin
2. build_apk.bat dosyasını çift tıklayın
3. APK dosyası build/app/outputs/flutter-apk/ klasöründe oluşacak

## Android Vale App  
1. C:\FunBreakVale_Android_Vale klasörüne gidin
2. build_apk.bat dosyasını çift tıklayın
3. APK dosyası build/app/outputs/flutter-apk/ klasöründe oluşacak

## Manuel Komut
Eğer batch dosyası çalışmazsa:
```
flutter build apk --release
```

## APK Dosya Konumları
- Customer: C:\FunBreakVale_Android_Customer\build\app\outputs\flutter-apk\app-release.apk
- Vale: C:\FunBreakVale_Android_Vale\build\app\outputs\flutter-apk\app-release.apk
